## 🎯 Règles d'or

1. **SMS = max 3 lignes**. Email = max 200 mots.
2. **Toujours une CTA claire** : RDV, réponse 1 ligne, lien à cliquer.
3. **Toujours un délai** : "vendredi 14h", "cette semaine", "avant le 25 mai".
4. **Toujours un bénéfice rapide** mentionné : 3 mois SEO, audit gratuit, place planning limité.
5. **Personnalisation minimale** : prénom + secteur + ville. Pas de copy-paste générique.
6. **Pas de superlatifs** ("incroyable", "extraordinaire") — fait amateur.
7. **Signature courte** : prénom + Linkeo + numéro. Pas de signature corporate ridicule.
