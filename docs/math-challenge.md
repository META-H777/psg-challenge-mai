## 🎯 Levier critique : taux de conversion RDV→vente

**Si on passe de 10% à 20%** (×2), on **divise par 2 le besoin RDV.**

| Closing rate | RDV pour 16 ventes | Effort |
|---|---|---|
| 10% | 160 | 🔴 Très intense (40/joueur) |
| 15% | 107 | 🟠 Intense (27/joueur) |
| **20%** | **80** | **🟢 Atteignable (20/joueur)** ✅ |
| 25% | 64 | 🟢 Confortable |

→ **Investir dans le closing rate via training + scripts** = ROI ×100 vs pousser plus de cold call.

## 🎯 Formule magique

```
Pour gagner = (RDV × Closing rate × Marge moyenne) > 120K€

Si closing rate = 18% et marge moy = 8K€ :
  RDV × 0.18 × 8000 > 120000
  RDV > 83

→ Cible 90+ RDV équipe sur le mois (≈ 23 RDV/joueur sur 4 semaines = 5-6 RDV/sem).
```
